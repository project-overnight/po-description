## Usage

- To seed the DB, run `npm run seed`
- api end point is localhost:6009/api/description/:id

## Requirements

An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

- Node 12.16.2
- mongo 4.2.5

## Development

⭐️ This project uses airbnb's(!) style guide.
